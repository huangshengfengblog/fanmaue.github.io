FanMa 中文名为 饭吗 ，名字为广西瑶族语系中一分支语系的译音，瑶话的语系有好几种，其实我也不知道自己是属于哪一语系的。瑶话各语系之间的语言各不想通，其他语系的瑶话我一点都听不同。

FanMa 博客采用了 VuePress 静态网站生成器搭来搭建，这是一个不需要服务后端的网站框架程序。由于 VuePress 官方的默认主题太过于简陋，于是我选择了 [VuePress Theme Hope](https://theme-hope.vuejs.press) 这款主题，这款主题内置有博客插件，可以让我更方便地去写博客。
